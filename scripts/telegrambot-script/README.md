# Telegrambot Script

## Edit telegrambot

Change `{Group ID Telegram}` & `{Bot Token}` on script to your environment

## Place telegrambot

Place telegrambot:
- Ubuntu = /usr/local/bin
- CentOS = /usr/bin

```
chmod +x telegrambot
mv telegrambot /usr/loca/bin
```

## Testing

```
telegrambot "testing"
```
